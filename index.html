Базові селектори:
Селектор за тегом - селектор елемента призначений для вибору елементів за ім'ям тега.
Селектор за класом - призначений для вибору елементів за класом (значенням атрибута class).
Селектор за ідентифікатором - призначений для вибору елемента за ідентифікатором (значенням атрибута id).
Селектор за атрибутом - призначений для вибора елементів за ім'ям атрибута та/або його значенням.
Універсальний селектор (*) - призначений для вибору усіх елементів.
